プライバシーポリシー

制定日: 2025年12月6日

TapTrans 運営事務局（以下「当事務局」といいます）は、本アプリにおける利用者の個人情報の取扱いについて、以下のとおりプライバシーポリシー（以下「本ポリシー」といいます）を定めます。

1. 収集する情報

本アプリは、サービスの提供にあたり、以下の情報を収集・利用することがあります。

  • ユーザーコンテンツ: 翻訳機能を利用するためにアップロードされたPDF内のテキストデータ、および生成された翻訳テキスト。
  • 端末情報: 認証およびセキュリティ維持のための端末識別子、OSバージョン、デバイスモデル名。
  • 購入情報: 有料プラン（サブスクリプション）の管理に必要な購入履歴、レシート情報、会員ステータス。
  • 利用状況ログ: サービスの品質向上や不正利用防止のためのアクセスログ、クラッシュレポート。

2. 利用目的

収集した情報は、以下の目的で利用します。

  1. AIによるPDF翻訳機能の提供のため。
  2. 有料プランの提供、課金管理、および契約状態の確認のため。
  3. 不正利用（スパム行為やスクリプト攻撃等）の防止およびセキュリティ確保のため。
  4. お問い合わせ対応および本アプリのバージョンアップ通知等のため。

3. 外部送信・第三者提供

本アプリは、翻訳およびAI機能を提供するために、利用者が入力したデータを以下の外部サービス事業者に送信します。

(1) Google LLC (Gemini API / Firebase)

  • 送信される情報: 翻訳対象のテキストデータ、認証用ID
  • 利用目的: AIによるテキスト生成、ユーザー認証、データベース保存
  • データの取扱い: GoogleのプライバシーポリシーおよびAPI利用規約に従い処理されます。なお、本アプリ経由でAPIに送信されたデータは、通常、AIモデルの学習には使用されません。

(2) RevenueCat

  • 送信される情報: 匿名のユーザーID、購入レシート情報
  • 利用目的: サブスクリプションのステータス管理

上記および法令に基づく場合を除き、利用者の同意なく第三者に個人情報を提供することはありません。

5. 免責事項

本アプリのAI翻訳機能によって生成されたテキストに個人情報が含まれていた場合でも、当事務局は一切の責任を負いません。翻訳対象のPDFには、他者のプライバシーを侵害する情報や、厳重な管理が必要な機密情報を含めないようご注意ください。
