Privacy Policy

Date of Enactment: December 6, 2025

The TapTrans Operation Team (hereinafter referred to as "the Team") establishes this Privacy Policy (hereinafter referred to as "this Policy") regarding the handling of users' personal information in this application.

1. Information We Collect

This application may collect and use the following information to provide services.

  • User Content: Text data within PDFs uploaded to use the translation function, and the generated translated text.
  • Device Information: Device identifiers, OS version, and device model name for authentication and security maintenance.
  • Purchase Information: Purchase history, receipt information, and membership status necessary for managing paid plans (subscriptions).
  • Usage Logs: Access logs and crash reports for improving service quality and preventing unauthorized use.

2. Purpose of Use

The collected information will be used for the following purposes:

  1. To provide AI-based PDF translation functions.
  2. To provide paid plans, manage billing, and confirm contract status.
  3. To prevent unauthorized use (such as spamming and script attacks) and ensure security.
  4. To respond to inquiries and notify about version updates of this application.

3. External Transmission and Provision to Third Parties

This application transmits data entered by users to the following external service providers to provide translation and AI functions.

(1) Google LLC (Gemini API / Firebase)

  • Information transmitted: Text data to be translated, authentication ID
  • Purpose of use: AI text generation, user authentication, database storage
  • Handling of data: Processed in accordance with Google's Privacy Policy and API Terms of Service. Note that data transmitted to the API via this application is generally not used for training AI models.

(2) RevenueCat

  • Information transmitted: Anonymous user ID, purchase receipt information
  • Purpose of use: Subscription status management

Except as described above or required by law, we will not provide personal information to third parties without the user's consent.

5. Disclaimer

The Team assumes no responsibility even if the text generated by the AI translation function of this application contains personal information. Please be careful not to include information that infringes on the privacy of others or confidential information requiring strict management in the PDFs to be translated.
